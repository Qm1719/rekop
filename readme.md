REKOP is a simple mobile app to track "points" between friends while playing poker or blackjack or even outside of card games.
You can add players, adjust their "points" per hand, navigate between hands, and save/load full match histories.
Features
Per-player balances with quick -5 / -1 / +1 / +5 buttons
Hands navigation (Previous hands edits affect forward to later hands)
Match history sidebar
Dark, compact UI
