### REKOP – Record Every Kind of Outcome as Points

REKOP is a simple mobile app to track “points” between friends while playing poker, blackjack, or even outside of card games.
You can add players, adjust their points per hand, navigate between hands, and save/load full match histories.

### Features
- Per-player balances with quick **-5 / -1 / +1 / +5** buttons  
- Hands navigation (edits to earlier hands ripple forward to later hands)  
- Match history sidebar with saved matches  
- Dark, compact UI
